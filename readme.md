# FaceRecAvoidBoss

- 一个用来玩的小玩具，随便写写，代码质量贼底

## 如何去玩

### Requirements

```shell
$ pip install -r requirements.txt
```
仅仅适配了windows，MacOS和Linux需要修改一下切屏的操作
需要电脑有摄像头，并且OpenCV编号0就能调用（废话） 不是的话需修改代码
### 录入数据

创建文件夹: ./faces 后，人脸对准摄像头，运行命令，然后等待弹出窗口后按```Q```退出

```shell
$ python Test.py
```

### For Fun!

运行命令，然后等等

```shell
$ python main.py
```
